# Contributing

We welcome contributions to the Physical AI Book project!

## Bug Fixes

- **Weekly Review:** All pull requests labeled with `bug` will be reviewed on a weekly basis, typically on Fridays.
- **Process:**
  1. Fork the repository.
  2. Create a new branch for your bug fix.
  3. Submit a pull request to the `main` branch.
  4. Add a clear description of the bug and the fix.
  5. Ensure all CI checks pass.
  6. The project maintainers will review the PR and may request changes.
  7. Once approved, the PR will be merged.

## Feature Requests

- Please open an issue with the `enhancement` label to discuss new features.

## Code of Conduct

Please note that this project is released with a Contributor Code of Conduct. By participating in this project you agree to abide by its terms.
